\## Code Snippets for Hands-On Sessions

In this repository, you will find the files you need for the hands-on session. 



1. **lps22hb**: the LPS22hb demo.

**2) lps22hb\_shell**:  this project should be modified to enable I2C and sensor Shell. See the corresponding slide.

**3) lps22hb\_sensorSim**: as described in the slide, the simulated sensor must be activated in the DeviceTree overlay file and the simSensor node must be selected in the code.

**4) lps22bh\_thread**: this project is identical to the lps22hb project. It should serve as a starting point and sensor handling should be implemented via thread.
